# Backend

## Requirements

- [git](https://git-scm.com/)
- [pnpm](https://pnpm.io/)
- [Node.js](https://nodejs.org/)

## Installation

```bash
pnpm install
```

## Running the app

```bash
# Development
pnpm start
```

```bash
# Watch mode
pnpm start
```

```bash
# Production mode
pnpm start
```

## Test

```bash
# Unit test
pnpm test
```

```bash
# E2E test
pnpm test:e2e
```

```bash
# Test coverage
pnpm test:cov
```
